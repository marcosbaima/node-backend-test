# Test Backend - Raks

> NodeJs API Restfull Test

This project is a test developement in NODEJS Restfull with TYPESCRIPT, EXPRESS, JEST, TYPEORM, MYSQL, DOCKER CONTAINER, SOLID   

![]()

## Installation

OS X & Linux:

sh
yarn 


Windows:

sh
yarn

Docker
sh
DOCKER start id container or name

TypeORM
sh

Start SERVER
sh
yarn dev:server
